Content of this Website

There are three kinds of users

  • Admin: Can access to each page of the website, he can create, read, update and delete any content, movies, comments, users ...
  • Registered user: He can access to the movies description registered in the database, read and post comments, he can edit his profile add favorite places, visualise it on a map, add movies to his liked movie list and he can rate a movie.
  • Visitor: Just access to the movie page
Purpose

Why I created this website

I created this website in order to practice my Angular skills. I learnt it, during my academic exchange in the University of Western Ontario, in Canada during the Winter Semester 2017. I realized a project based on the MEAN stack. M stands for MongoDb which is a noSQL database, the E stands for Express, it is a powerful Node Js framework, very useful to manage route. The A stands for Angular, a front end framework, by the way, I use Angular 4 for this website. And the last but not least, the N stands for Node Js the backend part of the project.
