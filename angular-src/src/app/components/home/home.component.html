Content of this Website

This website is based on IMDb, the purpose is to give the opportunity to anyone to add content to a movie description.

To access to the entire site you need to create an account.

Main Functionalities:

  • Look every page description of the movies available on the Database
  • Add a movie to the Database
  • To post a comment about a movie
  • To rate a movie
  • To like a movie
  • To modify some information about your profile
  • To filter the movies displayed on the movie tab
  • To look the profile of the others users

Tuto

Add a movie to the database: Go to the tab "POST A MOVIE". There are two possibilities: you can either fill a form with the information about the movie to add, or you can search a movie from IMDb and add it.

Look the movies registered on the database:Go to the tab "MOVIES", there is a list of every movie registered on the database. The main information is displayed, if you want to know more about a movies click on it. You will be redirect to the description of this movie.

Post a movie or add content to a movie: Go to the description of a movie. At the end of the page, there is a form, you have to give a title to the comment, then write your comment below. Once, the comment posted, you can delete it, at any time. To add content, go below and click on the button "Add Content". Then you have to choose the type of this content, "information" or "Section". An "information" will be displayed of the first part of the page, it should be a short information. A "section" is a full paragraph, it's longer than an information.

Like or rate a movie: Go to the description of a movie. On the right part of the screen, a section allows you to rate the movie and to like it. The average of this movie is also displayed.

To see your profile: Go the tab "PROFILE". There are four parts:

  • Bio: You can visualize your information, and also modify it
  • Like: You can visualize which movies you liked
  • Rating: You can visualize which movies you rated, and modify the rate
  • Places: This is currently unrelated to the website, on this part, you can add places that you have already seen and places that you want to see. Once added, you can displayed those places on a map.

Display statistics: Go to the tab "DASHBOARD". You can see, the number of movies registered on the database, the number of comments posted and the number of users registered. There is also, the Top 5 of the best rated movies and of the most rated movies. At the bottom of the page, the list of users with their statistics are displayed, to look their profile, click on the one that you want to see.

Technologies used


To create this web application, I used Angular 4 which is a powerful JavaScript framework. I also used Node JS and Express for the back end part. My database is MongoDb and I used Bootstrap 3 for the styles.

This web application is totally responsive, so it should look great on Tablet and Mobile ;)
It's also secure, the passwords are salted and hashed.

In order to get data of the movies, I used two APIs OMDb and TMDb. I also used Google GeoCoding API and Google Maps API for the section "places" on the profile page.
